## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio-website.git
```

Navigate to the project directory:

```bash
cd portfolio-website
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
